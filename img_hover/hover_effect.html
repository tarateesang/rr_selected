Dark overlay + image zoom (Work/Play section)

Work/Play section: this effect is done by placing a contained image and an overlay div inside a main container, which controls the hover effects of this component.

To make it look like the Work/Play section proposed, simply create and style more divs inside the parent container, except instead of them appearing on hover, make them visible at all times.
