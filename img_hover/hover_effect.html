Image hover effects

Dark overlay + image zoom (Work/Play section)

Work/Play section: this effect is done by placing a contained image and an overlay div inside a main container, which controls the hover effects of this component.

To make it look like the Work/Play section proposed, simply create and style more divs inside the parent container, except instead of them appearing on hover, make them visible at all times.

Image zoom only (images under work/play/popular)

The zoom-in effect was done by placing a contained image inside a container, which controls the effects when hovered over. Overall, it is the same component as the example above, just without the overlay div.

You can also zoom into the image by hovering over this text. As this zoom-in effect is done for only clickable article thumbnails , there will be an accompanying title link. This effect will not be present anywhere else in Robb Report.
  • For larger photos (i.e. this size) - zoom of 1.02 (2%) is good
  • For smaller photos, a zoom of 1.05 (5%) is better.
